FireSafe VR — Training
Find the safest exit. If safe, use the extinguisher using PASS.
Tip: smoke rises — stay low (crouch/crawl) in smoky areas.
Fire Safety Tips
  • Do not use elevators during a fire.
  • Stay low under smoke; crawl if needed.
  • Only use an extinguisher if the fire is small and you have a clear exit.
  • Follow EXIT signs and arrows; if one exit is blocked, find an alternative.
  • If clothes catch fire: Stop, Drop, and Roll.
