# FireSafe VR — AR/VR Fire Training Webpage (A-Frame)

This is a simple **VR-ready fire emergency training simulation** built with **HTML + JavaScript + A‑Frame**.

## How to run

- Open `index.html` in a modern browser (Chrome/Edge recommended).
- Click **Start Simulation**.
- Optional: click the **VR goggles** icon in the bottom-right of the scene to enter VR mode (if your device supports it).

## Controls

- **Move**: WASD  
- **Look**: Mouse  
- **Interact**: Click  
- **Extinguisher**:
  - Click the extinguisher to pick it up
  - Press **P** to pull the pin
  - Hold **Space** to spray (aim at the base of the fire)

## Training features included

- **Fire + smoke simulation** (smoke rises and thickens over time)
- **Smoke hazard awareness** (warnings if you stay upright in heavy smoke)
- **Emergency exits + guidance arrows**
- **Blocked exit scenario** (one exit is randomly blocked each run)
- **60-second timer challenge**
- **Safety tips** section
- **Quick quiz** section

## Files

- `index.html` — UI + A‑Frame scene
- `style.css` — HUD styling
- `app.js` — simulation logic (fire, smoke, extinguisher, exits, timer, quiz)
